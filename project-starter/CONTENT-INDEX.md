# Content Index

*A running list of every piece of content on the site. The purpose is dedup: before writing anything new, check this file so you don't accidentally write something you already have. Update this file every time you publish.*

---

## How to use this file

Every time Claude Code is about to write new content (blog post, landing page, comparison page, LinkedIn post, whatever), it should read this file first and confirm the topic is not already covered.

Add a `CLAUDE.md` rule: "Before writing any new content, check CONTENT-INDEX.md. If a similar topic exists, either (a) update the existing piece instead, or (b) propose a meaningfully different angle."
